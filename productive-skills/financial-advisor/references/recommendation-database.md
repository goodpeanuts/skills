# Financial Recommendation Database

This database provides structured recommendation templates organized by analysis type and scenario. Each recommendation includes observation templates, impact assessment, specific action steps, and priority levels.

---

## Recommendation Structure

Each recommendation follows this format:

```markdown
**Observation:** [What the data shows]
**Impact:** [Why this matters]
**Actions:** [Specific steps to take]
**Priority:** [Immediate/Short-term/Long-term]
**Timeline:** [Expected timeframe for implementation and results]
```

## Recommendation Guardrails (Use Before Any Template)

1. Treat all numeric targets in templates as baseline ranges, not mandatory fixed values.
2. Personalize targets by:
   - cash flow stability;
   - debt burden;
   - life stage and dependents;
   - cost-of-living region;
   - user risk tolerance and priorities.
3. Avoid command-only language when feasibility is uncertain. Provide at least one lighter alternative path.
4. Prefer neutral, non-judgmental wording. Describe tradeoffs and risks, not moral labels.
5. Use local resources for professional support (country/region appropriate), not a single country-specific default.

## Priority Framework

When multiple recommendations apply, prioritize by:

### Immediate (This Week)
- Cash flow negative
- Emergency fund <1 month
- Debt-to-asset >70%
- Credit utilization >70%
- Any "Critical" score component

### Short-term (1-3 months)
- Emergency fund 1-3 months
- High-interest debt payoff
- Budget establishment
- Insurance gaps

### Medium-term (3-12 months)
- Emergency fund 3-6 months
- Investment allocation
- Income diversification
- Major expense planning

### Long-term (1+ years)
- Retirement optimization
- Estate planning
- Advanced tax strategies
- Legacy planning

## Recommendation Formatting in Reports

When presenting recommendations:

1. **Number and categorize**
   ```markdown
   ### Recommendations

   #### Immediate Priority
   1. **[Recommendation Title]**
      - **Rationale:** [Why this matters]
      - **Action:** [Specific steps]
      - **Timeline:** [When to complete]
      - **Expected Impact:** [Quantified benefit]
   ```

2. **Show interconnections**
   - "Completing #1 will free up ¥[X] to accelerate #3"

3. **Provide roadmap**
   - Month 1: Recommendations #1, #2
   - Month 2-3: Recommendations #3, #4
   - Month 4-6: Recommendations #5, #6

4. **Track progress**
   - Set review dates
   - Define success metrics
   - Celebrate milestones
